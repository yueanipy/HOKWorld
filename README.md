# HOKWorld — 《王者荣耀世界》黑盒视觉自动化

靠屏幕画面识别加标准键鼠模拟做三件事:**自动钓鱼**(包括大鱼 QTE)、**剧情跳过**(剧情结束会误按，但是非致命问题，后续再解决)、**跑图自动采集**(经过材料/宝箱出现「F + 名称」提示就识别并按 F 采集,按手型图标区分该不该采)。

只截屏、只发标准键鼠输入,不读写游戏内存、不注入、不改封包、不碰任何游戏文件。仅供学习研究,使用风险自负,请遵守游戏用户协议。

> **想直接用、不装 Python?** 到 [Releases](https://github.com/yueanipy/HOKWorld/releases) 下载免装安装版(当前 `v0.3.3`:`HOKWorldScript-0.3.3-Setup.exe`),双击安装即用。下面是从源码运行的说明。

## 运行环境

- Windows 10 / 11(x64),Python 3.10 及以上
- 游戏以窗口或无边框窗口运行,并且保持前台不被遮挡

## 跑起来

```bat
git clone git@github.com:yueanipy/HOKWorld.git
cd HOKWorld
python -m venv .venv && .venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

运行中 **F12** 全局急停。

## 配置与名单(都存程序目录 `data\`)

配置、日志、采集黑/白名单都写在**程序目录下的 `data\`**(`config.json`、`blacklist.txt`、`whitelist.txt`、`logs\hokworld.log` 等)——随程序。`data\` 不在安装器的文件清单里，**卸载**(cmd + control控制面板或点击 uninstall)会连同程序一起清干净。

## 自动钓鱼

站在钓鱼点位上，确定处于钓鱼状态并且落点为蓝色，然后就可以先确定循环次数，点击开始即可开始钓鱼，后续加入暂停钓鱼，抛竿,等上钩,拉杆,收线,结算,接着续钓。大鱼会触发 QTE;到等级上限或连续空抛太多次会进行停机;结束打印一份简短的统计。

## 跑图自动采集

「实时检测」里的「经过材料自动采集(F)」**默认开启**,和剧情跳过一起跑:跑图经过材料/宝箱时游戏弹「F + 名称」提示,脚本实时认出并按 F 采。

- **按图标分类**:F 键帽确认有提示 → 看键帽右侧图标 —— 手型(采集)/ 重现图标就采,NPC / 商店 / 制作 / 对话 / 组队等别的图标不动。
- **碰撞名单(黑名单)**:图标也是采集手型、却不该采的交互(渡石、语印、捕获、启动滑索、松开滑索、启动冲云翼…)已**内置**,遇到自动跳过;误采了别的,到「设置 · 采集碰撞名单」加一行即可。
- **白名单(强制采)**:写进去的名字识别到就一定采(优先级最高,能盖过碰撞名单;也能强制采图标不是手型的东西)。
- 内置名单随版本更新自动生效;你自己增删的存程序 `data\`、更新非必要情况不会强制修改。

## 几点提醒

- 模板是从 1920 宽的真实客户端帧裁的,运行时每帧先归一化到 1920 宽再匹配。分辨率、画质、HUD 差太多时阈值可能要重标定,改 `fishing/matcher.py` 里的阈值和 ROI（也就是只适用于1080,1440,2160等16：9显示器）。
