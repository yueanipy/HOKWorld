# HOKWorld

《王者荣耀世界》的黑盒视觉辅助工具。

只看游戏画面，只发送正常键鼠输入。不读取游戏内存，不注入，不改封包，也不修改游戏内容。

## 须知

- 游戏必须在前台。切走以后任务会暂停，不会一直把游戏抢回前台。
- 自动启动只负责把游戏正常启动并进入一次，之后用户主动切换窗口，就由用户自己切回来。
- 游戏更新、界面缩放、HUD 调整或者画面差异过大，都可能让识别失效。
- 注意黄昏时候会大幅度增加漏浇并且降低自动寻路钓鱼可能性。

## 已完成

### 自动启动游戏

实时检测可选择自动启动游戏。游戏已经打开时会直接接着运行，不会重复启动。

启动流程会识别加载、公告、开始游戏和进入角色画面等状态，不按固定时间死等。进入游戏后才开始后续检测。

### 实时检测

实时检测启动后，只在需要操作时动作：

- 可跳过剧情时，打开确认并完成跳过。
- 普通对话、点击空白继续和黑屏过场会自动推进。
- 已经回到角色画面后立即停止剧情操作。
- 支持暂停、继续和停止。
- 没有触发内容时只保持低频检测，不做多余输入。

### 自动钓鱼

角色先站到可以钓鱼的位置，确认落点有效，再设置次数并开始。

程序会处理抛竿、等待上钩、拉杆、收线、结算和续钓，也会处理大鱼阶段的方向操作。达到目标数量、等级上限或者连续多次无法正常抛竿结束停止，并给出本次结果。

### 跑图自动采集

经过材料、宝箱或可重现目标时，游戏出现交互提示，程序会先判断图标和名称，再决定是否操作。

- 材料和宝箱等采集目标才会处理。
- NPC、商店、制作、对话、组队等其他交互默认不动。
- 名称没识别清楚时不按，避免把误识别当成采集。
- 已内置一批容易和采集图标混淆的交互，遇到会跳过。
- 用户可以自己增加强制采集名单和跳过名单，用户设置优先。

### 每日任务一条龙

每日任务可以按需勾选并调整执行顺序，目前包含：

- 农贸作物收割、播种和浇水，可选择第二列或第五列路线。
- 培养箱收割、播种和浇水。
- 宠物派遣、奖励领取和宠物喂食。
- 每日拍照、兴趣圈点赞和朝闻道奖励领取。
- 前往好友家累计完成两次浇水。
- 每日钓鱼三次。
- 自动前往荣耀塔完成烹饪或制药。

任务之间会自动返回角色画面并继续下一项；实时检测会在每日任务运行时暂停，全部结束后恢复。所有每日任务默认关闭，由用户自行选择需要执行的内容。

## 使用说明

1. 启动软件，并允许管理员权限。
2. 选择需要的功能。
3. 如果启用了自动启动游戏，直接点开始；没有启用就先让游戏处于前台。
4. 运行中切走游戏会暂停，切回游戏后再继续。
5. 任何时候出现不对，按 **F12** 急停。

自动钓鱼、实时检测和其他任务不会同时抢键鼠。一个任务运行时，其他任务需要等它结束。

## 采集名单怎么用

设置里有两类名单：

- **强制采集**：识别到就采，优先级最高。
- **碰撞跳过**：图标看起来像采集，但实际不希望程序操作的内容。

一行写一个名称即可。更新软件不会覆盖用户自己添加的内容。

如果某个交互被误按，就把它加入碰撞跳过；如果某个目标一直不采，再确认名称是否能识别，必要时加入强制采集。不要一开始把大量内容全部塞进强制采集。

## 数据与更新

用户设置、采集名单和运行日志都保存在本机。

覆盖更新会保留这些内容；正常卸载时会随程序一起清理。日志只用于排查运行状态，不会自动上传。

## 分辨率和画面要求

目前主要按 16:9 画面适配，常见的 1080p、1440p 和 2160p 都使用同一套相对坐标识别。

以下情况可能影响识别：

- 超宽屏或非 16:9 分辨率。
- 单独修改游戏 HUD 大小或位置。
- 画面被其他窗口、悬浮层或录屏提示遮挡。
- 极端滤镜、亮度或者游戏版本更新后界面变化。

遇到问题先恢复默认 HUD，并确认游戏画面没有被遮挡。
